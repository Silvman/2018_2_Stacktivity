# 2018_2_Stacktivity

## SPA

Сервер запускаем с помощью
```
npm i
npm start
```
А потом заходим на http://localhost:3000


## Шаблонизация
Используется Handlebars

Шаблоны лежат в директориях модулей

Для перекомпиляции шаблонов используется команда

```
npm run hand
```
